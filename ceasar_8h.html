static error_e split_lines_between_threads ( const HANDLE  input_file_handle,

)

Splits the file for farther proccessing by deviding its lines equally to the threads. When number of lines is not devisable by number of threads, the thread at the beginning will receive the extra load. That is, if we have 6 lines in the file, and 4 threads: thread_0: 2 lines thread_1: 2 lines thread_2: 1 lines thread_3: 1 lines.

The devision will be done by assiging start position and end position for each thread (inclusive). If a thread does not get any lines, in case there are less lines then threads, then end and start position will be set to -1.

[in]input_file_handle
[in]num_of_threads
[out]p_thread_params
