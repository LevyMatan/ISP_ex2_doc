Encrypts or decrypts a message from a file.

Parameters
[in]argc
[in]argvOn top of the function name the program takes a maximum of three more arguments. These parameters must appear in the following order:
  1. input file name string: holdes the full/relative path for the text file to be cyphered.
  2. encryption key: the key the program will use to encrypt or decrypt the file.
  3. number of threads: the number of threads to use to encrypt or decrypt the file.
  4. encrypt/decrypt flag: indication weather to encrypt or decrypt the file, the option are '-e' and '-d'
Returns
int On success the return value is 0 (ERROR_OK), a nonzero vlaue will be returned on failure, indicating the failure.
